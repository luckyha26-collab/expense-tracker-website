# Expense Tracker

## Documentation

### Overview
The Expense Tracker is a simple web application that allows users to track their expenses effectively. Users can add, edit, and delete their expenses, providing a clear overview of their spending habits.

### Features
- Add new expenses
- Modify existing expenses
- Remove expenses
- View total expenses

## Installation Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/luckyha26-collab/expense-tracker-website.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd expense-tracker-website
   ```
3. **Install the required dependencies**
   ```bash
   npm install
   ```
4. **Start the application**
   ```bash
   npm start
   ```
5. **Open your browser** and go to `http://localhost:3000` to use the Expense Tracker.

## Usage Guide
1. **Adding an Expense**:
   - Click on the 'Add Expense' button.
   - Fill in the required fields including the amount, category, and date.
   - Click 'Submit' to save the expense.

2. **Editing an Expense**:
   - Click on the expense you want to edit.
   - Modify the fields as needed.
   - Click 'Save' to apply changes.

3. **Deleting an Expense**:
   - Click on the trash icon next to the expense you wish to delete.
   - Confirm the deletion to remove the expense from the tracker.

4. **Viewing Total Expenses**:
   - The total expenses will be displayed at the top of the page, updating in real-time as you add or remove items.
